3. 코드 블록

pre 태그를 이용한 코드 블록 예시입니다:

def parse_html(content: str) -> dict:
    """HTML 문서를 파싱하여 구조화된 데이터로 반환합니다."""
    soup = BeautifulSoup(content, "html.parser")
    return {
        "title": soup.title.string if soup.title else "",
        "headings": [h.get_text() for h in soup.find_all(["h1", "h2", "h3"])],
    }

5. 단락 구조

첫 번째 단락입니다. 단락은 p 태그로 구분됩니다. 각 단락은 독립적인 텍스트 블록으로 파싱되어야 합니다.

두 번째 단락입니다. HTML 백엔드는 단락 사이의 공백을 올바르게 처리해야 합니다. 이 단락에는 여러 문장이 포함되어 있습니다. 문장이 길어져도 하나의 단락 블록으로 유지되어야 합니다.

세 번째 단락으로, 한국어 텍스트 처리를 검증합니다. 문서 처리 시 UTF-8 인코딩이 올바르게 유지되어야 하며, 특수 문자(예: &, <, >, ")도 올바르게 처리되어야 합니다.
